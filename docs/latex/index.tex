This project is a simple engine with an asteroid mini game attached.\hypertarget{index_Instructions}{}\doxysection{Instructions}\label{index_Instructions}

\begin{DoxyEnumerate}
\item This project was built on Windows\+OS.
\item You have to have SDL2, SDL\+\_\+image, SDL\+\_\+ttf, SDL\+\_\+mixer loaded in your environment.
\end{DoxyEnumerate}\hypertarget{index_Game}{}\doxysection{Commands}\label{index_Game}

\begin{DoxyEnumerate}
\item Use A/D key to let player one move left or right.
\item Use Left/\+Right Arrow key to let player two move left or right
\item Press space key to let player one shoot.
\item Press L key to let player two shoot.
\item When player win/lose the game, press ESC key to go back to main menu. 
\end{DoxyEnumerate}
